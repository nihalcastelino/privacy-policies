Privacy Policy

Splitword - Word chunks with one-way choices
Effective Date: April 20, 2026  |  Last Updated: April 20, 2026

Current Build Scope

Splitword is currently a local Flutter mobile puzzle app. The live implementation in this repository does not require sign-in, does not send gameplay data to FluffyLlama servers, and stores only local progress on your device.

What Data The Current App Stores

  • Unlocked level progress stored locally on your device.
  • The current in-progress board state so the app can resume after a relaunch on the same device.
  • No cloud account, payment, or ad profile data in the current build.

What The Current App Does Not Do

  • It does not create a Firebase account.
  • It does not upload puzzle progress to Firestore or any FluffyLlama backend.
  • It does not currently use RevenueCat, AdMob, analytics SDKs, or platform game services in this repository build.
If Splitword later ships with cloud save, analytics, ads, purchases, or platform leaderboards, this policy must be updated before release to reflect the real implementation.

Your Controls

You can clear the current app data by removing app storage on your device. Because the current build does not store account-linked data on FluffyLlama servers, there is no server-side account to delete at this stage.

Children and General Audience

Splitword is designed as a general-audience word puzzle. The current build is intended to avoid unnecessary personal data collection by default.

Contact and Future Updates

If the release scope changes, this page will be updated to match the real shipped stack and services. Read the matching delete-account page here: delete-account-splitword.html.
