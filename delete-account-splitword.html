Splitword - Current app account status
Effective Date: April 20, 2026  |  Last Updated: April 20, 2026

Current Status

The current Splitword repository build does not create online accounts or store account-linked gameplay data on FluffyLlama servers. It is a local Flutter mobile app that keeps only device-local progress.

How To Delete Current App Data

  1. Open app settings for Splitword on your device.
  2. Clear local app storage for Splitword.
  3. Reopen the app to confirm progress has been removed.

What This Removes

  • Unlocked level progress saved locally on device.
  • Any saved in-progress board state used for relaunch and resume on the same device.

What Does Not Exist In The Current Build

  • No Firebase account to delete.
  • No cloud save record to delete.
  • No purchase, ad, or leaderboard profile linked to a user account in the current repository build.
If Splitword later ships with cloud accounts or synced saves, this page must be replaced with the real account-deletion workflow before store release.
